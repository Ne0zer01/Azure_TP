### ðŸŒž CrÃ©er et configurer le bridge Linux :

```bash
  # crÃ©ation du bridge
  ip link add name vxlan_bridge type bridge

  # on allume le bridge
  ip link set dev vxlan_bridge up 

  # on dÃ©finit une IP sur cette interface bridge
  ip addr add 10.220.220.201/24 dev vxlan_bridge

  # ajout de l'interface bridge Ã  la zone public de firewalld

  # activation du masquerading NAT dans cette zone
  firewall-cmd --add-masquerade --permanent

  # on reload le firewall pour que les deux commandes prÃ©cÃ©dentes prennent effet
  firewall-cmd --reload
```

**CrÃ©ation du fichier *_setup_vxlan_bridge.sh_* :**

```bash
  sudo nano /usr/local/bin/setup_vxlan_bridge.sh
```

**CrÃ©ation du fichier *_vxlan.service_* :**

```bash

```

**Faire en sorte que le script *_setup_vxlan_bridge.sh_* s'exÃ©cute automatiquement au dÃ©marrage :**

```bash

  Description=Setup VXLAN interface for ONE

  [Service]
  Type=oneshot
  RemainAfterExit=yes
```
